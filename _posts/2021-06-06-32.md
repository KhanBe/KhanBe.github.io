## 레포지토리 생성 하기

- 인텔리제이 프로젝트 생성
- ctrl + shift + a 입력
- share project on github 클릭
- 깃허브 로그인
- 보통 .idea파일은 공유 안한다. (인텔리제이 자동생성파일이라, 체크 해제)

## 커밋 하기

- ctrl + k 누르면 커밋창 뜬다.
- commit messege 를 적는다.
- 커밋한다.

## 푸쉬 하기

- ctrl + shift + k 누르면 푸쉬 창이 뜬다.
- 푸쉬한다.
