## 문제

c언어를 해야할 일이 있어서 c언어 연습이 필요하다.

숫자 하나 입력하고 약수를 모두 출력하면 된다.

### 코드

```
#include <stdio.h>
int main() {
	int a;
	scanf("%d",&a);
	
	for(int i=1;i<=a;i++){
		if(a%i==0) printf("%d ",i);
	}
	return 0;
}
```
