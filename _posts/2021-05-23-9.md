자료구조 스택에 대해 정리 한다.


**개념**
1. 스택 접근 방법은 언제나 목록의 끝에서만 일어난다.
2. 스택은 한 쪽 끝에서만 자료를 넣거나 뺄 수 있는 선형 구조으로 되어 있다.
3. 후입선출 구조 (LIFO, Last-In-First-Out)구조 이다.(마지막 자료가 먼저 나감)

자바를 이용한 스택
- 선언
```java
Stack<T> stack = new Stack<T>()
```

- 데이터 삽입
```java
stack.push();
```

- 마지막 데이터 삭제
```java
stack.pop();
```

- 최근 들어온 데이터 출력
```java
stack.peek();
```

- 스택이 비었는지 반환
```java
boolean empty = stack.isEmpty();
```

- 스택 크기 반환
```java
int size = stack.size();
```
