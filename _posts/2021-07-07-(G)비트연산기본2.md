## 문제

두개의 수 a,b 입력받고

첫 줄에 a>>b

둘째 줄에 a<<b 출력한다.


### 코드

```
#include <stdio.h>
int main() {
	int a,b;
	
	scanf("%d",&a);
	scanf("%d",&b);
	
	printf("%d\n",a>>b);
	printf("%d",a<<b);
	return 0;
}
```
